What Will You Get From This Extension?

Helpful Debugging Suggestions. Some broad ideas of things to change/do, some oddly specific things to change/do, also some Motivational texts.

15 minute alarms, when you need a short break, so you don't loose track of time.

Links to developer-friendly sites like StackOverflow, GitHub, Reddit and CodePen and option of adding customized links as well.

Want Dodo the Duck in your Own system?

The Set-Up Is Easy. Follow the following steps -
Download the zip file and extract it, and then go to chrome://extensions
Enable Developer Mode, if you haven't.
Select Load Unpacked
Find the folder that contains all the files in this zip file and select it.
Ta-da! Dodo has arrived!
